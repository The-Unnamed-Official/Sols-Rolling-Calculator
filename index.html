Version v2 Update Notes

UI Rework & New Simulation Method

Updated 2026/07/20 at 4:30 PM CEST

Latest changes:

  • Complete interface rework One coordinated visual and responsive overhaul across the simulator. V2 UI
    • Masthead & navigationRebuilt the title bar into a clearer simulation header with a dedicated build control, creator, deployment, game-coverage, and system-status readouts.
    • Roll FeedReworked the fixed-height feed, live-follow control, search, tier-aware lookup, sorting, sharing layers, result styling, and exact-sequence presentation.
    • Simulation ConsoleRebuilt Roll Controls, method cards, split pause control, status feedback, multiplier rows, and centered the Bullet message.
    • Run ParametersReorganized potion loadouts, environment controls, and Environment Shortcuts into systematic, overflow-safe command modules.
    • Quick PresetsAdded bank navigation and cleaner preset cards, with hover/focus-only −/+ steppers and safe wrapping for long potion combinations.
    • Settings & popupsUnified Audio, Rolling Settings, quality preferences, aura filters, notices, confirmations, and the update archive into one responsive popup system.
    • Resources & supportRebuilt the Resource Hub, added animated NoLifers and Unnamed's RNG cards, Matrix repository effects, and fixed clipped support cards.
    • Credits directoryReplaced the long credits list with an expandable, searchable directory and instant contribution-type filters.
    • Phone & tablet polishHardened every major UI for narrow screens with adaptive grids, contained controls, compact forms, and safer popup and menu layering.
    • Quality defaultsNew and cleared-data users start with reduced glitch effects enabled while the remaining optional quality preferences stay disabled.
  • Two simulation methods:
    • Added a simulation-method selector with Unnamed's Simulation Method remaining the default and retaining its existing end-of-run result behavior.
    • Added Sol's-Like Simulation Method, which displays every aura live in its exact roll order while continuously advancing progress.
    • Both methods use the same luck, aura chance, potion, biome, and RNG calculations, so only processing and presentation change.
    • Switching methods preserves selected potions, quantities, stacking, biome, time, luck, and other simulation settings, and method changes are locked while rolling.
      • Credit goes to qqener for sharing this idea with me!
  • Live rolling, pause & cutscenes:
    • Sol's-Like rolls now enter the feed exactly once and in the precise order generated, in both Single Potion and Multiple Potion modes.
    • Added an attached pause/resume control beside Roll, with a centered icon and safe resume from the next unprocessed roll.
    • The live roll feed now keeps updating without forcing the scrollbar downward while you review earlier rolls. A fixed Follow newest rolls button appears only when scrolling is desynchronized, then jumps to the latest aura and restores automatic following.
    • Cutscenes now interrupt rolling at the exact aura position, pause the progress bar, and resume without skipped, duplicated, or reordered results. If Luminosity rolls before Equinox, their cutscenes play in that exact order; every cutscene aura follows this rule, including Monarch.
    • Added an always-visible Automatically pause rolling after cutscene switch for manually resuming after each Sol's-Like cutscene.
    • The Sol's-Like speed slider remains editable while rolling is manually paused, and the new speed applies from the next resumed roll without restarting or changing the RNG sequence.
    • Preserved the existing stacked-potion priority, including Oblivion, Pump King's Blood, Tutorial Potion, and all remaining potion types.
  • Filter consistency:
    • Sol's-Like Simulation Method now obeys both Filter Aura Tiers and Filter Auras while entries are rolling live.
    • Filtered auras are removed only from feed and cutscene presentation; rolls, RNG calls, potion processing, progress, totals, and statistics remain mathematically unchanged.
  • Rolling behavior & feed tools:
    • Replaced speed presets with a 1–1,000 rolls-per-second slider, ranging from one roll per second to one roll per millisecond.
    • Added a roll-feed search bar that filters by either aura name or tier name, such as Legendary.
    • Added a post-run Sort button beside the search bar for completed simulations in both Unnamed's and Sol's-Like methods. Unnamed's cycles between Rarity and Alphabetical, while Sol's-Like also includes Recent for its original live roll order.
  • RoE setting preservation:
    • Enabling Rune of Everything (RoE) no longer resets the selected biome or time of day.
